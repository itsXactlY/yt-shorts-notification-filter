# Privacy Policy for YT_PROTOCOL_FILTER

This extension is built with a focus on user sovereignty and privacy.

## Data Collection
- **No Personal Data:** This extension does not collect, store, or transmit any personal identification information.
- **No Telemetry:** There are no analytics, tracking, or background pings to external servers.
- **Local Execution:** All filtering and redirection logic is executed locally on your device.

## Data Storage
- **Local Storage:** The extension uses Chrome's `storage` API to save your preferences (e.g., filter status, whitelist). This data remains on your device and is not synced to any remote servers.

## Third-Party Access
- This extension does not share any data with third parties.

## Permissions
- **Host Permission (YouTube):** Needed solely to identify and remove Shorts elements from the UI and to perform URL redirections.
